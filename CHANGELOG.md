# Changelog

## [1.0.0] - Initial Release

### Added
- Lottery Machine block — right-click to open GUI
- Celestia Coin item — required to spin
- Five rarity tiers: Common, Uncommon, Rare, Epic, Omega
- TOML-driven reward pool — fully configurable without code changes
- Spinning wheel GUI with highlight animation
- Hopper automation — coins in via sides, rewards out via bottom
- JEI integration — reward categories visible per tier
- In-game config screen via Mods > Better Gamba > Config
- /reload support — config changes apply without restart
- Config validation at server start and on /reload
