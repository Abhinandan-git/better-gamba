A NeoForge 1.21.1 lottery machine mod. Place the Lottery Machine block,
insert coins, and win configurable rewards set by the modpack administrator.
 
## Requirements
 
- Minecraft Java Edition 1.21.1
- NeoForge 1.21.1
- JEI (optional — for recipe display)
 
## Installation
 
Drop the `.jar` into your `mods/` folder. Configure reward pools and coin cost
in `config/bettergamba-common.toml` after first launch.
